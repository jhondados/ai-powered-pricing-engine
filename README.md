# 💰 AI-Powered Dynamic Pricing Engine

[![Revenue](https://img.shields.io/badge/Revenue%20Uplift-%2B18%25-green)](.) [![SKUs](https://img.shields.io/badge/SKUs%20Managed-2.4M-blue)](.) [![Repricing](https://img.shields.io/badge/Repricing-Every%2015min-orange)](.)

> **AI pricing engine** managing 2.4M SKUs with demand elasticity models + RL optimization. Competitor prices scraped every 15 minutes. **+18% revenue uplift** and **-12% stockout rate**.

## 🏗️ Pricing Architecture
```
Demand Signal ──▶ Elasticity Model ──▶ RL Price Agent ──▶ Price Decision
Competitor Price ──▶ Position Analysis ──▶ Rules Engine ──▶ Guardrails Check
Inventory Level ──▶ Urgency Score ──▶ Markdown Logic ──▶ Final Price
```
